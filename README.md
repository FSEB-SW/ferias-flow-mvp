# FeriasFlow — MVP

**Pos-Graduacao em Engenharia de Software — PUC-Rio**
Sprint de Gestao Agil de Projetos e Produtos — entrega de MVP (ideacao e planejamento).

## O produto

**FeriasFlow** e uma aplicacao web para **planejamento e aprovacao de ferias de equipes**. Centraliza,
num unico fluxo, o pedido do colaborador, o saldo de dias, o calendario da equipe e a decisao do
gestor — resolvendo a dor de aprovar ferias "no escuro", sem enxergar saldo, sobreposicao e
cobertura da operacao no mesmo lugar.
